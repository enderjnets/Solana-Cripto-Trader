## VEREDICTO DEL CEO

> **OPCIÓN B — Precios por segmento** con elementos de C (investigación básica del negocio)
>
> Un precio único deja dinero sobre la mesa en B2B y asusta a micro-negocios. Precios 100% individualizados por negocio son imposibles de escalar con agentes automatizados. La solución óptima: **3 segmentos bien definidos**, cada uno con su tabla de precios, y los agentes identifican el segmento en 2–3 preguntas clave.

---

## 1. POR QUÉ NO A LAS OTRAS OPCIONES

### ❌ Opción A — Precio único para todos
- **iCapelli Salon** pagaría lo mismo que **Cristina Ocando** (CLX Global / EVEM Data)
- Un salón de $25K/mes de ingreso y una empresa de contratos de $50K/evento tienen ROI completamente diferente
- Precio único óptimo para salones (~$200–$300/mes) significa regalar el servicio a corporativos
- Precio único óptimo para corporativos (~$800–$1,200/mes) aleja a pequeños negocios
- **Resultado: pierdes en ambos extremos**

### ⚠️ Opción C — Personalización total por negocio
- Requiere investigación profunda antes de cotizar → ralentiza el pipeline
- Los agentes tomarían decisiones de precio inconsistentes
- Crea negociación caso por caso → caos de gestión con 20+ prospectos
- **Funciona para deals de $5K+/año, no para SMB en volumen**

## 4. GUÍA PARA LOS AGENTES — Cómo Identificar el Segmento

### Las 4 Preguntas del Segmentador

Cuando un agente revisa un prospecto nuevo, debe responder estas 4 preguntas:

```
1. ¿Cuántos empleados tiene el negocio?
   → 1 persona o solopreneur       → MICRO
   → 2-10 empleados                → LOCAL
   → 10+ empleados / empresa       → SMB/CORP

2. ¿Maneja contratos o deals de alto valor?
   → No (servicios de $30-$300)   → MICRO o LOCAL
   → Sí (deals de $1K–$100K+)    → SMB/CORP

3. ¿Tiene múltiples negocios o líneas?
   → No                           → Un solo agente
   → Sí (2+ negocios)            → Bundle SMB (Multi-Negocio)

4. ¿Opera en un área premium o tiene marca establecida?
   → Sí (Cherry Creek, LoDo, dental especializado, med spa)
   → Subir al plan siguiente (Local Profesional → Local Premium)
```

### Árbol de Decisión Visual

```
¿Es B2B o tiene deals >$5K?
    │
    SÍ → SEGMENTO 3 (SMB/CORP)
    │     • 1 negocio → Standard $597/mes
    │     • 2+ negocios → Bundle $800/mes
    │
    NO ↓
    
¿Tiene 1 persona o es solopreneur?
    │
    SÍ → SEGMENTO 1 (MICRO)
    │     • Micro Básico $197/mes, $0 setup
    │
    NO ↓
    
→ SEGMENTO 2 (LOCAL)
    ¿Opera en zona premium o servicio especializado?
        SÍ → Local Premium $397/mes, $299 setup
        NO → Local Profesional $297/mes, $199 setup
```

### Señales de upgrade (Profesional → Premium)

Un negocio LOCAL sube a Premium si tiene **2+ de estas señales:**
- Ubicación en Cherry Creek, LoDo, Wash Park, o Downtown
- Es un "med spa", dental especializado, o clínica con lista de espera
- Tiene sistema de booking que necesita integración (Vagaro, Mindbody, etc.)
- Tiene 5+ reseñas mencionando "no pude comunicarme" en Google Reviews
- El dueño mencionó que perdió clientes por no contestar llamadas
