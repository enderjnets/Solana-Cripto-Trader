ALWAYS mix animated video clips (Wan2.1 I2V) WITH static photos/images (Ken Burns effect) in alternating pattern for all BitTrader videos. NEVER use only one type of visual content.

**How to apply:** When producing any video (short or long), alternate every 3-5 seconds between animated video clips and Ken Burns static photos.
